Privacy Policy for Message Tracker Extension

Last Updated: January 15, 2025

Summary: This extension does NOT collect, store, or transmit any personal data. All message counts are stored locally on your device only.

1. Information We Do NOT Collect

The Message Tracker Extension does NOT collect, access, read, or store:

  • Message content or text
  • Email addresses or contact information
  • Personal identifiable information (PII)
  • Browsing history or web activity
  • Location data
  • Authentication credentials (passwords, tokens, etc.)
  • Any data from websites other than Gmail and LinkedIn

2. What the Extension Does

The Message Tracker Extension:

  • Counts messages only: Tracks the number of messages sent and received on Gmail and LinkedIn
  • Stores counts locally: All data is stored in your browser's local storage (chrome.storage.local)
  • Generates CSV reports: Creates a simple CSV file with message counts (numbers only)
  • Sends email reports: Uses Gmail API to send the CSV report to your email address

3. Data Storage

All data is stored locally on your device using Chrome's storage API. The extension stores:

  • Daily message counts (numbers only)
  • Your name (for CSV report header)
  • Your email address (for sending reports)
  • Report send time preference

No data is sent to external servers or third parties.

4. Permissions Explained

The extension requires the following permissions:

  • storage: To save message counts and settings locally on your device
  • alarms: To schedule daily automatic email reports
  • identity: To authenticate with Gmail API for sending email reports
  • mail.google.com: To count Gmail messages by monitoring the page (content is never read)
  • linkedin.com: To count LinkedIn messages by monitoring the page (content is never read)
  • googleapis.com: To send email reports via Gmail API

5. Third-Party Services

The extension uses:

  • Gmail API: Only to send email reports. Google's privacy policy applies to Gmail API usage.

No other third-party services are used.

6. Data Sharing

We do NOT share, sell, or transmit any data to third parties. The only data transmission is:

  • Sending the CSV report to your own email address via Gmail API

7. Data Security

All data is stored locally on your device using Chrome's secure storage API. The extension:

  • Does not transmit data to external servers
  • Does not use analytics or tracking tools
  • Does not collect telemetry data

8. User Control

You have full control over your data:

  • You can view all stored data in the extension popup
  • You can delete all data by uninstalling the extension
  • You can configure or disable daily reports at any time

9. Children's Privacy

This extension does not knowingly collect any information from children under 13 years of age.

10. Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date at the top of this page.
